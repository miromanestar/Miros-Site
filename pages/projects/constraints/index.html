A simple visualizer for constraint-based programming, done in JavaScript!

About the segmentation simulator?

Constraint-based programming is a paradigm by which a the values of certain variables are dependent upon the values of other variables. For example, a constrainted variable might be a variable which must always be twice the size of another variable, and thus would automatically recalculate should that variable change. To model this, I've created a simple application which allows you to create basic mathematical expressions and connect them in a visual to demonstrate how this style of programming might actually work.
